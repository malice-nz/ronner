<div align="center">Universal and simple toast notification library</div><br>
<h1></h1>
<p><b>Ronner</b> is a recreation of the popular toast library: <a href="https://github.com/emilkowalski/sonner">Sonner</a>; with loads of customisation.</p>
<img alt="Example" src="https://github.com/user-attachments/assets/6c74993e-8ee7-4278-bad7-d55703cbd2c6" />
<h1></h1>
You can run Ronner using the <code>main.lua</code>, or, the official shortlink.

### GitHub
```lua
local Ronner = loadstring(game:HttpGet'https://ronner.malice.nz')':3'
```

### Shortlink
```lua
local Ronner = loadstring(game:HttpGet'https://ronner.malice.nz')':3'
```

<h1></h1>
Heres a small example

```lua
local Ronner = loadstring(game:HttpGet'https://ronner.malice.nz')':3'

Ronner.Init();

Ronner.Toast('Hello roblox!');
Ronner.Toast('Hello from ronner!');
Ronner.Toast('Hello from <copyable>malice.nz</copyable>!');
```
